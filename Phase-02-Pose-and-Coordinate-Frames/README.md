# Phase 02 — Pose and Coordinate Frames

Understand how robots represent position, orientation, coordinate frames, and transforms.

## Topics

- Robot pose
- Position and orientation
- Coordinate frames
- Transform trees
- ROS2 TF and TF2
- Map, odom, base_link, and sensor frames

## Deliverables

- Notes on pose and coordinate frames
- Diagrams of robot frames
- Examples of transform relationships
- Reflections on how coordinate frames support localization and mapping
