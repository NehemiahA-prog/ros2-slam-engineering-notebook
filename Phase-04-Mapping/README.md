# Phase 04 — Mapping

## Goal

Understand how robots use sensor data to build maps of their environment.

## Topics

- Occupancy grid maps
- LiDAR scan data
- Map resolution
- Static vs dynamic obstacles
- Map quality
- Saving and loading maps in ROS2

## Deliverables

- Mapping notes
- Example maps
- Screenshots from RViz
- Analysis of map quality and limitations
