# Phase 06 — SLAM

## Goal

Understand how localization and mapping are solved together when the robot does not already have a map.

## Topics

- Simultaneous Localization and Mapping
- Loop closure
- Sensor noise
- Drift correction
- LiDAR-based SLAM
- SLAM performance evaluation

## Deliverables

- SLAM notes
- Maps generated from real or simulated runs
- RViz screenshots
- Analysis of what worked, what failed, and why
